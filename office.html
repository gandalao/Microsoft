  • 我可以使用Microsoft 365 获取哪些Office应用？

    可获取 Word、Excel、PowerPoint、Outlook、OneNote、OneDrive 等 Office 应用。

    详细了解 Microsoft 365 计划：

    • 家庭版
    • 商业版
    • 企业版
  • Office 365和Microsfot365有何区别？

    Microsoft 365 包含你所知道的 Office 365 中的所有内容。Microsoft 365 旨在帮助人们和企业利用创新的 Office 应用、智能云服务和世界一流的安全性实现更多目标。

  • 可获取哪些Microsfot365计划？

    Microsoft 365 家庭版计划

    • Microsoft 365 家庭版
    • Microsoft 365 个人版

    Microsoft 365 商业版计划

    • Microsoft 365 商业基础版
    • Microsoft 365 商业标准版
    • Microsoft 365 商业高级版
    • Microsoft 365 商业应用版

    Microsoft 365 企业版计划

    • Microsoft 365 E3
    • Microsoft 365 E5
    • Microsoft 365 F1
    • Microsoft 365 企业应用版
  • 现有的 Office 365 家庭版和商业版订阅有什么变化？

    一些 Office 365 订阅将自动更名为 Microsoft 365 订阅。你无需执行任何操作。

    • Office 365 个人版将更名为 Microsoft 365 个人版。
    • Office 365 家庭版将更名为 Microsoft 365 家庭版。
    • Office 365 商业协作版将更名为 Microsoft 365 商业基础版。
    • Office 365 商业高级版将更名为 Microsoft 365 商业标准版。
    • Microsoft 365 商业版将更名为 Microsoft 365 商业高级版。
    • Office 365 商业版将更名为 Microsoft 365 商业应用版。
    • Office 365 专业增强版将更名为 Microsoft 365 企业应用版。

    以下 Office 365 企业版计划不会发生任何变化：

    • Office 365 E1
    • Office 365 E3
    • Office 365 E5
  • Microsoft 365 的价格更高吗？

    不，订阅费用将保持不变。

  • 哪些 Office 365 计划仍可用？

    以下 Office 365 企业版计划仍可用：

    • Office 365 E1
    • Office 365 E3
    • Office 365 E5
  • Microsoft 365 是否需要 Internet 连接？

    是，安装和激活 Microsoft 365 以及获取所有最新版应用和服务都需要 Internet 连接。

    请注意，如果你已经是订阅者，则无需重新安装或购买其他订阅。

    无需连接到 Internet 即可使用 Word、Excel 和 PowerPoint 等 Office 应用，因为这些应用程序是完全安装在计算机上的。
